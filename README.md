# Ecommerce Data Analysis Project - SQL and Python

## Project Overview
This project demonstrates end-to-end data analysis on an Ecommerce dataset using SQL and Python.  
The objective was to simulate real-world data analyst tasks such as database creation, data cleaning, querying, visualization, and business insight generation.

This project is suitable for entry-level Data Analyst portfolios.

---

## Project Objectives
- Analyze monthly order trends
- Calculate total revenue and KPIs
- Identify top customers by spending
- Perform SQL joins and aggregations
- Create Python visualizations
- Generate business insights from data

---

## Tools and Technologies Used
- MySQL Workbench - Database creation and SQL queries
- Python - Data analysis
- Pandas - Data manipulation
- Matplotlib - Data visualization
- Jupyter Notebook - Analysis environment
- GitHub - Project hosting

## Key Insights
- Order volume increases steadily over months
- Revenue is strongly linked to delivered orders
- A small percentage of customers contribute large revenue
- Seasonal trends are visible in monthly order data

## How to Run the Project

### SQL
1. Open MySQL Workbench
2. Create database
3. Import CSV datasets
4. Run SQL queries from ecommerce_queries.sql

### Python
1. Open Jupyter Notebook
2. Install required libraries:
   pip install pandas matplotlib
3. Open ecommerce_analysis.ipynb
4. Run all cells
